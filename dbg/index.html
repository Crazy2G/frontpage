Discord Banner Generator


Discord (which is a messaging platform built for gamers), offers "widgets", which are animated images of your server by their API, with the server's icon and the number of members. To make this work, you need to enable the widget on your Discord Server (Server Configurations > Widget), insert the server ID (enable the developer mode on Discord, right click your server and press "Copy ID"), choose the type of widget you'd like to use, and bam!
